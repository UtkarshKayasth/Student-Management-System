About This Project

The Student Management System is a Django-based web application designed to simplify student record handling at Skyline Public School.

Key Features:

  • Create, Read, Update, and Delete (soft delete) student records
  • Clean UI with navigation and form validation
  • Extensible design for restoring deleted records (optional)

Developer Info:

Developed by Utkarsh Kayasth as a mock project to practice Django fundamentals like CRUD operations, template inheritance, static files, and views.
